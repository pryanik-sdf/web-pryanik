# Webmin-like Utility

Веб-утилита для управления Ubuntu Linux, аналогичная Webmin.

## Описание

Эта утилита предоставляет веб-интерфейс для:
- Просмотра списка установленных пакетов
- Удаления пакетов
- Подключения к терминальной консоли

Доступ заблокирован, требуется авторизация под учетной записью администратора.

## Установка и запуск на Ubuntu

1. Установите Node.js и npm на Ubuntu:
   ```
   sudo apt update
   sudo apt install nodejs npm
   ```

2. Скопируйте проект на сервер.

3. Установите зависимости:
   ```
   npm install
   ```

4. Запустите сервер:
   ```
   npm start
   ```

Сервер будет доступен по адресу http://localhost:3000.

## Аутентификация

- Логин: admin
- Пароль: пароль администратора сервера (измените в server.js на чтение пароля root или настройте auth)

## Примечания

- Терминал использует временную реализацию на exec; на Ubuntu рекомендуется раскомментировать node-pty для полноценной интерактивной консоли.
- Для node-pty требуется Python 3.6+ для компиляции. На Ubuntu установите python3-dev перед npm install.
- Команды apt-get требуют sudo; настройте NOPASSWD для пользователя в /etc/sudoers:
  ```
  sudo visudo
  ```
  Добавьте строку: `youruser ALL=(ALL) NOPASSWD: /usr/bin/apt-get, /usr/bin/dpkg-query` (замените youruser на имя пользователя).
- Или запускайте сервер под root.

## Безопасность

- Измените пароль администратора в server.js.
- Используйте HTTPS в продакшене.
- Запускайте под ограниченным пользователем с sudo NOPASSWD.

## Функциональность

- **Пакеты**: Просмотр и удаление установленных пакетов с помощью dpkg-query и apt-get.
- **Консоль**: WebSocket-соединение для выполнения команд (упрощенная версия).
